# Jewel-War-IA
World War Jewel (V2) – RTS 2D tri-faction com IA integrada. Três times com squads de 3 (engenheiro/assaltante/batedor) disputam joias ou dominação de base em mapas procedurais. UI em pygame (menus + treino), IA heurística + self-play headless, respawn, construção (núcleo/torres/paredes) e ambiente Gym para RL.
